# Что это за приложение
Это Android-приложение для поиска работы, которое подключается к API с вакансиями. Пользователь может искать вакансии по ключевым словам, применять фильтры (место работы, отрасль, зарплата), сохранять понравившиеся вакансии в избранное и просматривать их даже без интернета. В приложении есть светлая и темная тема.

# Какие технологии использовал
Архитектура построена на MVVM с разделением на слои data, domain и presentation. Для асинхронной работы использовал корутины.

Из библиотек: Room для хранения избранных вакансий с полными данными (описание, навыки, контакты сохраняются в БД для офлайн-доступа), Retrofit с OkHttp для запросов к API вакансий, Gson для сериализации JSON, Koin для внедрения зависимостей.

Для интерфейса использовал ViewBinding, Navigation Component для навигации между экранами, BottomNavigationView для переключения между главным экраном, избранным и экраном команды. RecyclerView с адаптерами и пагинацией, список вакансий подгружается по мере скролла. Картинки загружаю через Glide с закруглёнными углами и placeholder-изображениями.

Фильтры настраиваются через отдельные экраны: выбор страны и региона (двухуровневый список), выбор отрасли с поиском, указание зарплаты и чекбокс «Не показывать без зарплаты». Кнопка фильтра на главном экране меняет цвет при активных фильтрах.

Поиск работает с дебаунсом (задержка 2 секунды между вводом символов, чтобы не спамить запросами). Реализована пагинация — при скролле вниз подгружаются следующие страницы с обработкой ошибок (если интернет пропал, показывается Toast, а уже загруженные вакансии остаются на экране).

Избранные вакансии хранятся в локальной базе данных Room. При добавлении в избранное сохраняется вся информация: описание, навыки, контакты, логотип. Благодаря этому вакансию можно открыть и полностью просмотреть даже без интернета.

Детальный экран вакансии показывает название, зарплату, компанию и логотип, город, требуемый опыт, занятость и график, описание с HTML-форматированием, список ключевых навыков, контакты (имя, email, телефон). Есть кнопка «Поделиться» — ссылка на вакансию отправляется через стандартный Intent. Из контактов можно позвонить или написать письмо — тоже через системные интенты.

# Что умеет приложение
Искать вакансии по ключевым словам через API (с дебаунсом, чтобы не спамить запросами). Подгружать новые вакансии при скролле (пагинация).

Фильтровать вакансии по стране и региону (двухуровневый выбор), отрасли (с поиском), зарплате. Включать опцию «Не показывать без зарплаты». Фильтры сохраняются между сессиями в SharedPreferences.

Добавлять вакансии в избранное и удалять из избранного. Просматривать избранные вакансии полностью без интернета (описание, навыки, контакты доступны офлайн).

Открывать детальную информацию о вакансии: название, зарплата, компания и логотип, город, опыт, занятость/график, описание, навыки, контакты. Делиться ссылкой на вакансию через стандартный системный Intent. Звонить по телефону из контактов и отправлять письмо на email.

Обрабатывать ошибки: при отсутствии интернета показывается плейсхолдер «Нет интернета», при ошибке подгрузки следующей страницы — Toast, а список не исчезает. Если вакансия не найдена — плейсхолдер «Вакансия не найдена». Если по запросу ничего нет — плейсхолдер «Таких вакансий нет».

Переключать светлую и темную тему — выбор темы автоматически подхватывается системой через ресурсы night/notnight.
